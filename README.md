## What is Donna?

Donna is a small functional programming language focused on:

- 🌸 Developer joy
- ⚡ Native compilation
- 🧩 Simple tooling
- ✨ Clean syntax
- ⚙️ Great interoperability through FFI

Donna is self-hosted and targets native binaries through QBE.

## Packages

### Command line
- [argparse](https://github.com/donna-lang/argparse) — Argument parser for Donna.

### Compilers
- [donna](https://github.com/donna-lang/donna) — The Donna programming language compiler and CLI.
- [stdlib](https://github.com/donna-lang/stdlib) — Donna standard library.

### Cryptography
- [crypt](https://github.com/donna-lang/crypt)— Cryptographic hashing utilities for Donna

### Parsing & Formats

- [markdown](https://github.com/NikolasSkyl/markdown) — Minimum Markdown support for Donna.
- [parsetoml](https://github.com/donna-lang/parsetoml) — TOML parser.

### Templates

- [mustache](https://github.com/NikolasSkyl/mustache) — Bindings for mustache template language

### Testing
- [unittest](https://github.com/donna-lang/unittest) — Unit testing library for Donna.

### Web

- [leaf](https://github.com/NikolasSkyl/leaf) — Static site generator written in Donna.
- [mongoose](https://github.com/NikolasSkyl/mongoose) — HTTP/web tooling experiments.
- [toucan](https://github.com/NikolasSkyl/toucan) — A micro web framework


## Editor support

- [vscode-donna](https://github.com/NikolasSkyl/vscode-donna) — VS Code extension for Donna
- [donna-zed](https://github.com/NikolasSkyl/donna-zed) — Zed Editor Extension for Donna Language
